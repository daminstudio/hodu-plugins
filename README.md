# Hodu Plugins

Official plugins for [Hodu](https://github.com/daminstudio/hodu) ML toolkit.

## Official Plugins

| Plugin | Type | Description |
|--------|------|-------------|
| [aot-cpu](./hodu-backend-aot-cpu-plugin) | Backend | AOT compiler for CPU via C code generation. Supports cross-compilation and embedded targets. |

## Installation

```bash
hodu plugin install aot-cpu
```
